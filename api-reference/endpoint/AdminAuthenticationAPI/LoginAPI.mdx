# Admin Login API

Login endpoint for admin users.

## Endpoint
`POST http://develop.trip.stage.dcs/clients/csb/public/api.php/auth/login`

## Description
Authenticates an admin user by submitting a username and password using `form-data`.

## Request

### Method
`POST`

### Body Parameters (form-data)
| Name     | Type   | Description         |
|----------|--------|---------------------|
| Username | string | Admin email address |
| Password | string | Admin password      |

#### Example Form Data
```json
{
  "Username": "",
  "Password": ""
}
```

> ⚠️ **Note:** This is test data. Never expose real credentials in production documentation.
